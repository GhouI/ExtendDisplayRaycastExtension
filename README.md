# Extend Displays

Raycast extension for Windows that switches the current display mode to `Extend these displays`.

## What it does

Running the `Extend Displays` command launches Windows' built-in `DisplaySwitch.exe /extend`.

## Local development

1. Run `npm install`
2. Run `npm run dev`
3. Open the extension in Raycast and trigger `Extend Displays`

## Note

The `author` field in `package.json` must be a real Raycast username for the Raycast CLI to validate the project. If you plan to publish or keep the metadata accurate, replace it with your own Raycast username.
